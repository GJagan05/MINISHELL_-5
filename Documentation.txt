OBJECTIVE :
__________
1.To design and implement a custom Linux shell that:
2.Executes internal commands
3.Executes external commands
4.Supports pipes
5.Handles signals (Ctrl+C, Ctrl+Z)
6.Manages background processes
7.Maintains job control using linked list
8.Demonstrates Linux internals concepts



CONCEPTS(Linux Internals):
__________________________
->This project demonstrates the following core Linux concepts:

Concept	                            Used For
________                          _______________
fork()	                            Create new process
execvp()	                        Execute external commands
wait() / waitpid()	                Wait for child process
WNOHANG	                            Non-blocking child check
WUNTRACED	                        Detect stopped process
signal()	                        Handle signals
kill()	                            Send signal to process
pipe()	                            Inter-process communication
dup2()	                            Redirect file descriptors
getenv()	                        Environment variables
Linked List	                        Job control storage

WORKING FLOW :
_____________
Step 1: Start Shell
        Display prompt minishell$
        Wait for user input

Step 2: Read Input
        User enters command
        Command is parsed

Step 3: Identify Command Type
        If Built-in → execute internally
        If External → fork() + execvp()
        If Invalid → print error

Step 4: Handle Signals
        Ctrl+C → terminate foreground process
        Ctrl+Z → stop process & store in linked list
        SIGCHLD → clean finished background processes

SIGNAL HANDLING :
_________________
Ctrl + C (SIGINT)

--> Terminates foreground process

--> Does NOT terminate shell


Ctrl + Z (SIGTSTP)

--> Stops process

-->Inserts into linked list


SIGCHLD

--> Detects background process completion

USES:

--> waitpid(-1, &status, WNOHANG);



JOB CONTROL (Linked List):
_________________________
STRUCTURE:

typedef struct node
{
    pid_t pid;
    char cmd[100];
    struct node *link;
} NODE;

FUNCTIONS:

        insert_at_first()
